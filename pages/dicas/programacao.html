Dicas de Programação

Boas práticas, atalhos e conceitos essenciais para estagiários de desenvolvimento.

1. Use Git desde o primeiro dia

Versione tudo. Crie branches por feature, faça commits pequenos e com mensagens claras. Isso facilita revisões e evita perda de trabalho.

git checkout -b feature/login
git add .
git commit -m "Adiciona tela de login"
git push origin feature/login
GitIniciante

2. Nomes claros valem mais que comentários

Prefira calcularSalarioLiquido() a calc(). Código se explica melhor com bons nomes do que com comentários explicativos.

Boas práticas

3. Aprenda a depurar, não só codar

Use o debugger do VS Code, coloque breakpoints e inspecione variáveis. Saber ler o stack trace economiza horas.

VS CodeDebug

4. Atalhos do VS Code que salvam tempo

Ctrl + P        -> ir para arquivo
Ctrl + Shift + P -> paleta de comandos
F12             -> ir para definição
Alt + Seta     -> mover linha
Ctrl + /        -> comentar/descomentar
VS CodeProdutividade

5. Python: virtualenv para cada projeto

Isole dependências para evitar conflitos entre projetos.

python -m venv venv
source venv/bin/activate   # Linux/Mac
pip install -r requirements.txt
